Amelia Laundry adalah aplikasi web berbasis Django yang dibuat untuk mengelola proses laundry secara digital.  
Sistem ini memiliki tiga jenis pengguna: Admin, Customer, dan Driver, yang masing-masing memiliki dashboard dan fitur berbeda.

---

## 1. Fitur Utama

### 1.1 Login & Registrasi
- Login berdasarkan role (Admin, Customer, Driver)
- Registrasi untuk customer
- Validasi input

### 1.2 Admin Dashboard
- Melihat seluruh pesanan
- Mengelola data customer & driver
- Melihat laporan transaksi
- Memperbarui status pesanan

### 1.3 Customer Dashboard
- Membuat pesanan laundry
- Melihat status pesanan
- Melihat estimasi biaya

### 1.4 Driver Dashboard
- Melihat pesanan yang harus dijemput / diantar
- Mengubah status pesanan
- Melakukan proses pengiriman
